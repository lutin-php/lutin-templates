Welcome to Your New Blog!

This is your first blog post. The blog-static starter is a simple, flat-file blog system that uses HTML files for content.

How It Works

Posts are stored as HTML files in the data/posts/ directory. Each file should be named with the format:

YYYY-MM-DD-post-slug.html

The date prefix helps with sorting, and the slug becomes part of the URL.

Writing Posts

You can use any HTML tags in your posts:

  • Use <p> for paragraphs
  • Use <h1>, <h2>, <h3> for headings
  • Use <strong> and <em> for emphasis
  • Use <code> for inline code
  • Use <pre><code> for code blocks
  • Use <ul> and <li> for lists
